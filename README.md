# Herbamed Blockchain

Plataforma digital descentralizada para el registro, validación y comercialización de plantas medicinales mexicanas.

## Descripción

Herbamed es una DApp (Aplicación Descentralizada) construida sobre la red Stellar utilizando Soroban para contratos inteligentes. El proyecto busca preservar y digitalizar el conocimiento tradicional herbolario mexicano, combinando la validación científica con la accesibilidad moderna de la tecnología blockchain.

## Estructura del Proyecto

```
herbamed-blockchain/
├── contracts/         # Contratos inteligentes en Rust/Soroban
│   ├── src/          # Código fuente de contratos
│   └── tests/        # Tests de contratos
├── frontend/         # Aplicación web Vue.js
├── backend/          # APIs y servicios adicionales
└── docs/            # Documentación del proyecto
```

## Tecnologías

- Soroban: Platform de contratos inteligentes de Stellar
- Stellar: Red blockchain principal
- Rust: Lenguaje para contratos inteligentes
- Vue.js: Framework frontend
- WSL2: Entorno de desarrollo Linux en Windows

## Configuración del Entorno de Desarrollo

1. Instalar WSL2 y Ubuntu
2. Instalar Rust y Cargo
3. Instalar Soroban CLI
4. Configurar entorno de desarrollo
